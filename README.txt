====How to run the project ?====. 

make sure that you have a python on your computer and an IDE recommended Pycharm.
download all the files :
	asteroid.py
	asteroids_main.py
	torpedo.py
	ship.py
	screen.py
put all files in one folder "AsteroidGame".
open this file in pycharm and run asteroids_main.py
note : you can change the number of asteroid by supply paramter.
to supply parameter go to => edit configuration => parameters => apply
*************
if you just want to play the game download asteroids_main.exe 
put this file in the same place where the mentioned files placed and run it.
